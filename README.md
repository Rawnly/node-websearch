## Installation

You must use **npm**:

```bash
  npm install -g node-websearch
```

## Usage

```bash
  # Search on Google
  # you can also use -g instead of --google
  websearch --google 'NodeJs'

  # Search on Wikipedia
  # you can also use -w instead of --wikipedia
  websearch --wikipedia 'Albert Einstein'
```
